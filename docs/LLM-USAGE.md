# 大模型调用说明

## 1. 模型信息

| 项目 | 说明 |
|------|------|
| 模型 | **agnes-2.0-flash** |
| API 地址 | `https://apihub.agnes-ai.com/v1` |
| 鉴权方式 | Bearer Token（Header: `Authorization: Bearer <key>`） |
| 超时时间 | 15 秒 |

## 2. 使用场景

### 2.1 AI 辅助分类（异常上报页）

**文件：** `src/lib/llm.ts` → `classifyWithLlm()`

**触发时机：** 用户在 `/tickets/new` 页面填写异常描述后，点击"AI 辅助分类"按钮

**Prompt 设计：**
- system prompt：`你是一个物流品控专家助手，只返回 JSON。`
- user prompt：包含 10 种可选异常类型及其中文标签，要求模型按指定 JSON 格式返回 `{"type":"异常类型","severity":"严重度"}`

**返回格式：**
```json
{"type":"lost","severity":"critical"}
```

### 2.2 AI 审批建议（工单详情页）

**文件：** `src/lib/llm.ts` → `getApprovalSuggestionWithLlm()`

**触发时机：** 审批人打开工单详情页时自动生成

**Prompt 设计：**
- system prompt：`你是一个物流审批专家，只返回 JSON。`
- user prompt：传入异常类型、异常金额、异常描述，要求模型返回 `{"suggestion":"审批建议","reason":"理由"}`

**返回格式：**
```json
{"suggestion":"建议核准赔付，同时安排补发","reason":"丢件属于高严重度异常，且金额在合理赔付范围内"}
```

## 3. 降级方案

| 场景 | 处理方式 |
|------|---------|
| LLM API 超时（15s） | 降级到关键词匹配 |
| LLM API 返回非 JSON | 降级到关键词匹配 |
| LLM API 鉴权失败 | 降级到关键词匹配，不影响主流程 |
| 网络异常 | 降级到关键词匹配 |

### 关键词匹配降级逻辑

**文件：** `src/lib/ai-classifier.ts`

- 预定义了 10 种异常类型的触发关键词（如"丢件/丢失/遗失"→ lost）
- 根据匹配到的关键词计算置信度（60%-100%）
- 审批建议基于异常类型有固定的推荐意见

## 4. "AI 建议，需人工确认"原则

| 场景 | 标注方式 |
|------|---------|
| 异常上报页 | 蓝色提示框顶部显示"🤖 AI 建议 — 需人工确认"，下方显示置信度和匹配关键词 |
| 工单详情页 | 审批操作区上方显示"🤖 AI 建议审批意见 — 需人工确认"，下方显示依据说明 |
| 最终决策 | AI 的建议不会自动执行，用户必须手动选择异常类型（可修改）并提交 |
